# Magic: The Gathering Cards REST API

A RESTful API for managing Magic card data using Node.js, Express, PostgreSQL

## Tech Used

- Node
- Express
- PostgreSQL
- node-postgres
- Postman

## Features

- CRUD operations to manage cards collection
- PostgreSQL database integration

## Installation

1. Clone the repository:
```bash
git https://github.com/adrian-guzman-301/techtonica-assignments/tree/main/rest-api-project
cd rest-api-project
```

2. run "npm install" in terminal

3. Start server: type "npm run server" in terminal

Server: `http://localhost3001`
